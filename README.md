# Valentine's Day Interactive Page

A romantic React component for a Valentine's Day proposal.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Interactive Valentine's proposal
- Evasive "No" button that escapes clicks/hovers
- Smooth animations with Framer Motion
- Responsive design with TailwindCSS
- Heartfelt message on acceptance

## Tech Stack

- React 18
- Next.js 14
- Framer Motion
- TailwindCSS
